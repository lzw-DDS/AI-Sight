🤖 AI Sight 工作原理

        
      

      

        

          
1

          

            
理解 Agent

            
接收用户创作需求，通过结构化分析拆解为可执行任务单元，识别关键词、风格倾向、输出格式要求。

          

        

        

          
2

          

            
生成 Agent

            
根据任务单元，针对每个 AI 工具（Midjourney / Kling / Suno）的特性，生成专用提示词，支持多版本对比。

          

        

        

          
3

          

            
执行 Agent

            
模拟完整工具链调用流程，可视化展示从提示词输入到最终产出的全过程，支持导出全部提示词。

          

        

        

          
💡 AI Sight 为纯前端工具，零 API 成本，所有分析均在浏览器本地完成。
